# NettoyageGmail

Application de nettoyage de boîte Gmail perso.

## Description

NettoyageGmail aide à organiser et supprimer les e-mails indésirables d'une boîte Gmail personnelle.

## Installation

1. Cloner le dépôt
2. Installer les dépendances

## Utilisation

Lancer le script principal pour démarrer l'application.
